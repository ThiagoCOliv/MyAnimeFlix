Inicio Séries Filmes Sobre

O MyAnimeFlix é uma ideia feita com base em um desafio de projeto do Bootcamp "JavaScript Game Developer" da plataforma DIO (Digital Innovation One) se baseando na interface do Netflix, porém decidi usar a temática de anime e criar duas páginas a mais do que foi desenvolvido no desafio como forma de aprimorar minhas habilidades com HTML, CSS e JavaScript.
